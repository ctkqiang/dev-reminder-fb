# Dev Reminder (Femboy Edition) 💖

> 一个温柔可爱的 VS Code 扩展，用伪娘风格轻声提醒你：该喝水、休息、活动啦～ 🌈


---

## 🧠 灵感来源

程序员不该一直坐着写代码不动！这个插件每隔一段时间，就会用温柔、撒娇、甚至小傲娇的方式弹窗提醒你：

- 💦 亲亲～喝口水嘛？
- 🧘‍♀️ 伸个懒腰吧，不然小蛮腰要塌啦～
- 🐾 快走两步！你不是沙发马铃薯！

适合 Z 世代、乙女脑、卷不动的打工人，以及需要有人“轻声威胁”你健康的人类。

---

## ✨ 功能特点

- ⏱️ **定时提醒**：默认每 45 分钟提醒一次（可调）
- 🎀 **多种语气**：撒娇、温柔、小傲娇，轮着来
- 🌸 **软萌文案**：中文提示，带 Emoji，小可爱风十足
- 🖥️ **开源透明**：你可以随意查看、修改提醒内容～

## 📦 安装方式

### 方法一：从 Marketplace 安装

👉 [点击安装](https://marketplace.visualstudio.com/items?itemName=ctkqiang.dev-reminder-fb)

### 方法二：从源码打包

```bash
git clone https://github.com/ctkqiang/dev-reminder-fb.git
cd dev-reminder-fb
npm install
npm run compile
vsce package
code --install-extension dev-reminder-fb-*.vsix
```

## 🧸 Q\&A

> ❓ 为什么要搞这个插件？

因为打工人真的太不健康了，身体提醒没用，需要一点带情绪价值的提示 ✨

> ❓ 会不会太娘/太软？

可以自己换文案！甚至可以选“强硬模式”：「起来！你坐得太久了，再不动我把你电脑关了。」

> ❓ 会不会打扰我写代码？

提醒是非强制的，轻轻弹窗一下，随手一关也没负担 💅
